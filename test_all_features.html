Feature Testing

1. Bot Mode Test

Steps:
1. Enter username
2. Click "Play vs Bot" button
3. Board should appear with your name vs Bot
4. Click on cells to make moves
5. Bot should respond after 1 second
6. Game should detect win/draw
Open Game

2. Multiplayer Test

Steps (need 2 browser windows):
1. Open game in 2 separate windows
2. Enter different usernames
3. Both click "Find Match" (Classic mode)
4. Should connect and show both player names
5. Players should take turns making moves
6. Moves should sync between windows
Open Game Window 1 | Open Game Window 2

3. Timer Mode Test

Steps:
1. Select "Timed" from game mode dropdown
2. Click "Find Match"
3. Timer should appear showing 30 seconds
4. Timer should count down during your turn
5. Timer should reset after each move
6. Game should end if time runs out
Open Game

4. Logout Test

Steps:
1. Play a game (bot or multiplayer)
2. Click "Logout" button
3. Should return to username screen
4. Game state should be reset
5. Should be able to login again
Open Game

5. Rematch Test

Steps (multiplayer only):
1. Complete a multiplayer game
2. Click "Request Rematch" button
3. Opponent should see rematch request
4. When both accept, new game starts
Open Game
